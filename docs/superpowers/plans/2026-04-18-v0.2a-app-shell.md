# v0.2a — App Shell & Editor — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a SwiftUI macOS app (`LithePGApp`) that connects to Postgres, runs SQL in a syntax-highlighted editor, and renders rows in a basic table — first shippable slice of v0.2.

**Architecture:** Three SPM targets — `LithePGCore` (driver + config + SSH + result model, UI-free), `lithepg` (existing CLI, unchanged surface), `LithePGApp` (new SwiftUI exe). Single persistent `PostgresConnector` per window. Runestone + tree-sitter SQL grammar for the editor. Pure SPM; no `.xcodeproj`.

**Tech Stack:** Swift 6.2, SwiftUI, `@Observable`, PostgresNIO 1.21+, Runestone (+ tree-sitter SQL), NSViewRepresentable bridge, Swift Testing.

**Spec:** `docs/superpowers/specs/2026-04-18-v0.2a-app-shell-design.md` — read §4–§10 before starting.

```swift
import Foundation

public enum ErrorRedaction {
    /// Stringifies the error via `String(describing:)`, then scrubs `password` substrings.
    public static func redactCredentials(in error: Error) -> String {
        redactCredentials(in: String(describing: error))
    }

    /// Scrubs `password` occurrences of the form `password: "x"`, `password=x`, `Password = x`, etc.
    public static func redactCredentials(in raw: String) -> String {
        let pattern = #"(password\s*[:=]\s*)("[^"]*"|[^",\s)]+)"#
        guard let regex = try? NSRegularExpression(pattern: pattern, options: .caseInsensitive) else {
            return raw
        }
        let range = NSRange(raw.startIndex..<raw.endIndex, in: raw)
        return regex.stringByReplacingMatches(in: raw, range: range, withTemplate: "$1[redacted]")
    }
}
```

```swift
import Foundation

public struct QueryResult: Sendable, Equatable {
    public let columns: [Column]
    public let rows: [Row]
    public let rowCount: Int
    public let elapsed: Duration
    public let status: Status
    public let truncated: Bool

    public init(
        columns: [Column],
        rows: [Row],
        rowCount: Int,
        elapsed: Duration,
        status: Status,
        truncated: Bool
    ) {
        self.columns = columns
        self.rows = rows
        self.rowCount = rowCount
        self.elapsed = elapsed
        self.status = status
        self.truncated = truncated
    }

    public struct Column: Sendable, Equatable {
        public let name: String
        public let typeName: String
        public init(name: String, typeName: String) {
            self.name = name
            self.typeName = typeName
        }
    }

    public struct Row: Sendable, Equatable, Identifiable {
        public let id: Int
        public let cells: [Cell]
        public init(id: Int, cells: [Cell]) {
            self.id = id
            self.cells = cells
        }
    }

    public enum Cell: Sendable, Equatable {
        case null
        case text(String)
    }

    public enum Status: Sendable, Equatable {
        case rows
        case command(tag: String, affected: Int)
        case empty
    }
}
```

- [ ] **Step 3: Implement `open`, `close`, and `execute` on `PostgresConnector`**

Open `Sources/LithePGCore/PostgresConnector.swift`. Add these members inside `public actor PostgresConnector` (after the existing state but before `runSelect1`):

```swift
    private var held: (connection: PostgresConnection, tunnel: SSHTunnel?)?

    public enum ExecuteError: Error, Equatable {
        case notConnected
        case alreadyConnected
    }

    public func open(config: ConnectionConfig) async throws {
        guard held == nil else { throw ExecuteError.alreadyConnected }
        let (effectiveHost, effectivePort, tunnel) = try await resolveTransport(config: config)
        let pgConfig = PostgresConnection.Configuration(
            host: effectiveHost,
            port: effectivePort,
            username: config.username,
            password: config.password,
            database: config.database,
            tls: try makeTLS(for: config)
        )
        do {
            let connection = try await PostgresConnection.connect(
                on: eventLoopGroup.next(),
                configuration: pgConfig,
                id: 1,
                logger: logger
            )
            held = (connection, tunnel)
        } catch {
            await tunnel?.close()
            throw error
        }
    }

    public func close() async {
        guard let current = held else { return }
        held = nil
        try? await current.connection.close()
        await current.tunnel?.close()
    }

    public func execute(_ sql: String) async throws -> QueryResult {
        guard let current = held else { throw ExecuteError.notConnected }
        let start = ContinuousClock.now
        let stream = try await current.connection.query(
            PostgresQuery(unsafeSQL: sql),
            logger: logger
        )

        var columns: [QueryResult.Column] = []
        var rows: [QueryResult.Row] = []
        var truncated = false
        let cap = 10_000

        for try await row in stream {
            if columns.isEmpty {
                columns = row.columns.map { col in
                    QueryResult.Column(name: col.name, typeName: String(describing: col.dataType))
                }
            }
            if rows.count >= cap {
                truncated = true
                break
            }
            var cells: [QueryResult.Cell] = []
            cells.reserveCapacity(row.count)
            for cell in row {
                cells.append(Self.renderCell(cell))
            }
            rows.append(QueryResult.Row(id: rows.count, cells: cells))
        }

        let elapsed = ContinuousClock.now - start
        let status: QueryResult.Status
        if let meta = stream.commandTag, !columns.isEmpty == false, rows.isEmpty {
            // `CommandTag` text is e.g. "INSERT 0 3", "UPDATE 5", "SELECT 2".
            status = Self.parseCommandTag(meta) ?? .command(tag: meta, affected: 0)
        } else if columns.isEmpty && rows.isEmpty {
            status = .empty
        } else {
            status = .rows
        }

        return QueryResult(
            columns: columns,
            rows: rows,
            rowCount: rows.count,
            elapsed: elapsed,
            status: status,
            truncated: truncated
        )
    }

    /// Parses a Postgres command tag like "INSERT 0 3" or "UPDATE 5" into a Status.
    /// PostgresNIO returns the raw tag string; this splits on whitespace and takes
    /// the last numeric token as the affected-row count.
    static func parseCommandTag(_ tag: String) -> QueryResult.Status? {
        let parts = tag.split(separator: " ").map(String.init)
        guard let verb = parts.first else { return nil }
        let affected = parts.reversed().first(where: { Int($0) != nil }).flatMap(Int.init) ?? 0
        return .command(tag: verb, affected: affected)
    }

    static func renderCell(_ cell: PostgresCell) -> QueryResult.Cell {
        guard let bytes = cell.bytes else { return .null }
        // Prefer text-format decoding; fall back to raw bytes via hex.
        if let s = try? String(fromPostgres: cell) {
            return .text(s)
        }
        return .text("<\(bytes.readableBytes) bytes>")
    }
```

```swift
import Foundation
import Observation
import LithePGCore

@Observable
@MainActor
public final class AppState {
    public enum ConnectionState: Equatable {
        case disconnected
        case connecting
        case connected(label: String)
    }

    public var connectionState: ConnectionState = .disconnected
    public var editorText: String = "" {
        didSet {
            if lastError != nil { lastError = nil }
        }
    }
    public var lastResult: QueryResult?
    public var lastError: String?
    public var isRunning: Bool = false

    public init() {}

    public func markConnecting() {
        connectionState = .connecting
    }

    public func markConnected(label: String) {
        connectionState = .connected(label: label)
        lastError = nil
    }

    public func markDisconnected() {
        connectionState = .disconnected
        lastResult = nil
        isRunning = false
    }

    public func markRunning() {
        isRunning = true
        lastResult = nil
        lastError = nil
    }

    public func markIdle() {
        isRunning = false
    }

    public func setResult(_ result: QueryResult) {
        lastResult = result
        lastError = nil
    }

    public func setError(_ message: String) {
        lastError = message
    }

    public func clearError() {
        lastError = nil
    }
}
```

**Goal:** Wire `AppState` to `PostgresConnector`. These are async methods that perform real I/O; they get unit-test coverage for state transitions around mocked success/failure where practical, and integration coverage via a later manual smoke.

- [ ] **Step 1: Extend `AppState` with async connect/disconnect/runQuery**

Append these members inside the `AppState` class body (after the existing sync helpers):

```swift
    /// Active connector for the lifetime of a connection. nil when disconnected.
    private var connector: PostgresConnector?

    /// Currently-running query Task. Non-nil only while isRunning == true.
    private var runningTask: Task<Void, Never>?

    public func connect(_ config: ConnectionConfig) async {
        markConnecting()
        let c = PostgresConnector()
        do {
            try await c.open(config: config)
            connector = c
            let label = "\(config.username)@\(config.host):\(config.port)/\(config.database)"
            markConnected(label: label)
        } catch {
            try? await c.shutdown()
            markDisconnected()
            setError(ErrorRedaction.redactCredentials(in: error))
        }
    }

    public func disconnect() async {
        runningTask?.cancel()
        runningTask = nil
        if let c = connector {
            await c.close()
            try? await c.shutdown()
            connector = nil
        }
        markDisconnected()
    }

    public func runQuery() {
        guard let c = connector, !isRunning else { return }
        let sql = editorText
        guard !sql.trimmingCharacters(in: .whitespacesAndNewlines).isEmpty else { return }
        markRunning()
        runningTask = Task { [weak self] in
            defer { Task { @MainActor in self?.runningTask = nil; self?.markIdle() } }
            do {
                let result = try await c.execute(sql)
                await MainActor.run { self?.setResult(result) }
            } catch is CancellationError {
                await MainActor.run { self?.setError("Query cancelled.") }
            } catch {
                await MainActor.run {
                    self?.setError(ErrorRedaction.redactCredentials(in: error))
                }
            }
        }
    }

    public func cancelRunningQuery() {
        runningTask?.cancel()
    }
```

```swift
import SwiftUI
import Runestone

struct EditorView: NSViewRepresentable {
    @Binding var text: String

    func makeNSView(context: Context) -> TextView {
        let tv = TextView()
        tv.text = text
        tv.showLineNumbers = true
        tv.backgroundColor = .textBackgroundColor
        tv.insertionPointColor = .textColor
        tv.editorDelegate = context.coordinator
        return tv
    }

    func updateNSView(_ nsView: TextView, context: Context) {
        if nsView.text != text {
            nsView.text = text
        }
    }

    func makeCoordinator() -> Coordinator {
        Coordinator(parent: self)
    }

    final class Coordinator: NSObject, TextViewDelegate {
        var parent: EditorView
        init(parent: EditorView) { self.parent = parent }

        func textViewDidChange(_ textView: TextView) {
            parent.text = textView.text
        }
    }
}
```

```swift
import LithePGCore

struct ConnectSheet: View {
    @Environment(AppState.self) private var state

    @State private var url: String = ""
    @State private var tlsEnabled: Bool = false
    @State private var tlsCAPath: String = ""
    @State private var sshEnabled: Bool = false
    @State private var sshTarget: String = ""
    @State private var showCAImporter: Bool = false

    var body: some View {
        VStack(alignment: .leading, spacing: 12) {
            Text("Connect to Postgres").font(.title2).bold()

            Form {
                TextField("postgres://user:pass@host:5432/db", text: $url)
                    .textFieldStyle(.roundedBorder)
                    .disableAutocorrection(true)

                Toggle("Use TLS (verify-full)", isOn: $tlsEnabled)
                    .disabled(sshEnabled)

                if tlsEnabled {
                    HStack {
                        TextField("CA .pem path", text: $tlsCAPath)
                            .textFieldStyle(.roundedBorder)
                        Button("Choose…") { showCAImporter = true }
                    }
                }

                Toggle("Use SSH tunnel", isOn: $sshEnabled)
                    .disabled(tlsEnabled)

                if sshEnabled {
                    TextField("user@bastion.example.com:22", text: $sshTarget)
                        .textFieldStyle(.roundedBorder)
                        .disableAutocorrection(true)
                }
            }

            if let error = state.lastError, state.connectionState == .disconnected {
                Text(error)
                    .font(.callout)
                    .foregroundStyle(.red)
                    .textSelection(.enabled)
            }

            HStack {
                Spacer()
                Button(connectButtonTitle) {
                    Task { await attemptConnect() }
                }
                .keyboardShortcut(.defaultAction)
                .disabled(!canConnect)
            }
        }
        .padding(20)
        .frame(minWidth: 480)
        .onAppear(perform: applyEnvAutofill)
        .fileImporter(
            isPresented: $showCAImporter,
            allowedContentTypes: [.x509Certificate, .pemPath].compactMap { $0 },
            allowsMultipleSelection: false
        ) { result in
            if case .success(let urls) = result, let first = urls.first {
                tlsCAPath = first.path(percentEncoded: false)
            }
        }
    }

    private var connectButtonTitle: String {
        state.connectionState == .connecting ? "Connecting…" : "Connect"
    }

    private var canConnect: Bool {
        if state.connectionState == .connecting { return false }
        if url.isEmpty { return false }
        if tlsEnabled && tlsCAPath.isEmpty { return false }
        if sshEnabled && sshTarget.isEmpty { return false }
        return true
    }

    private func applyEnvAutofill() {
        let env = ProcessInfo.processInfo.environment
        if url.isEmpty, let v = env["POSTGRES_URL"] { url = v }
        if tlsCAPath.isEmpty, let v = env["POSTGRES_TLS_CA"] {
            tlsCAPath = v
            tlsEnabled = true
        }
        if sshTarget.isEmpty, let v = env["POSTGRES_SSH"] {
            sshTarget = v
            sshEnabled = true
        }
    }

    private func attemptConnect() async {
        do {
            let base = try ConnectionConfig(url: url)
            let ssh = sshEnabled ? try Self.parseSSH(sshTarget) : nil
            let config = ConnectionConfig(
                host: base.host,
                port: base.port,
                database: base.database,
                username: base.username,
                password: base.password,
                tlsMode: tlsEnabled ? .verifyFull : .disable,
                pinnedRootCertificatePath: tlsEnabled && !tlsCAPath.isEmpty ? tlsCAPath : nil,
                sshConfig: ssh
            )
            await state.connect(config)
        } catch {
            state.setError(ErrorRedaction.redactCredentials(in: error))
        }
    }

    static func parseSSH(_ raw: String) throws -> ConnectionConfig.SSHConfig {
        let parts = raw.split(separator: "@", maxSplits: 1).map(String.init)
        guard parts.count == 2, !parts[0].isEmpty else {
            throw NSError(domain: "ConnectSheet", code: 1,
                          userInfo: [NSLocalizedDescriptionKey: "SSH target must be user@host[:port]"])
        }
        let hp = parts[1].split(separator: ":").map(String.init)
        let host = hp[0]
        let port = hp.count == 2 ? (Int(hp[1]) ?? 22) : 22
        return .init(host: host, port: port, user: parts[0])
    }
}

extension UTType {
    fileprivate static var pemPath: UTType? { UTType(filenameExtension: "pem") }
}
```

```swift
import SwiftUI
import LithePGCore

struct ResultsTable: View {
    let result: QueryResult

    var body: some View {
        VStack(spacing: 0) {
            if result.truncated {
                Text("Showing 10,000 of ≥10,000 rows. Pagination lands in v0.2b.")
                    .font(.callout)
                    .foregroundStyle(.secondary)
                    .frame(maxWidth: .infinity, alignment: .leading)
                    .padding(.horizontal, 8)
                    .padding(.vertical, 4)
                    .background(Color.yellow.opacity(0.2))
            }
            switch result.status {
            case .command(let tag, let affected):
                statusLine(
                    "\(tag) — \(affected) row\(affected == 1 ? "" : "s") affected (\(formatElapsed(result.elapsed)))"
                )
            case .empty:
                statusLine("Empty result (\(formatElapsed(result.elapsed))).")
            case .rows:
                rowsTable
            }
        }
    }

    @ViewBuilder
    private var rowsTable: some View {
        Table(result.rows) {
            TableColumnForEach(result.columns.indices, id: \.self) { colIdx in
                TableColumn(result.columns[colIdx].name) { (row: QueryResult.Row) in
                    Text(display(row.cells[colIdx]))
                        .textSelection(.enabled)
                }
            }
        }
    }

    private func display(_ cell: QueryResult.Cell) -> String {
        switch cell {
        case .null: return "NULL"
        case .text(let s): return s
        }
    }

    private func statusLine(_ text: String) -> some View {
        Text(text)
            .font(.callout.monospaced())
            .frame(maxWidth: .infinity, alignment: .leading)
            .padding(.horizontal, 8)
            .padding(.vertical, 6)
    }

    private func formatElapsed(_ d: Duration) -> String {
        let ms = Double(d.components.seconds) * 1000 + Double(d.components.attoseconds) / 1e15
        return String(format: "%.0f ms", ms)
    }
}
```

Note: SwiftUI `Table` requires statically-known columns in most versions. If `TableColumnForEach` is unavailable on the target macOS version, fall back to a `List` + `LazyVGrid` rendering. This is a v0.2a compromise documented in the commit message.

```swift
        @Bindable var bindableState = state

        VSplitView {
            VStack(spacing: 0) {
                toolbar
                EditorView(text: $bindableState.editorText)
            }
            .frame(minHeight: 160)

            VStack(spacing: 0) {
                if let error = state.lastError {
                    ErrorBanner(
                        message: error,
                        onReconnect: reconnectIfConnectionLost(error: error),
                        onDismiss: { state.clearError() }
                    )
                }
                if let result = state.lastResult {
                    ResultsTable(result: result)
                } else if state.isRunning {
                    ProgressView("Running…")
                        .progressViewStyle(.circular)
                        .frame(maxWidth: .infinity, maxHeight: .infinity)
                } else {
                    Text("Results will appear here.")
                        .foregroundStyle(.secondary)
                        .frame(maxWidth: .infinity, maxHeight: .infinity)
                }
            }
            .frame(minHeight: 160)
        }
    }

    private var toolbar: some View {
        HStack(spacing: 8) {
            Button(action: { state.runQuery() }) {
                Label("Run", systemImage: "play.fill")
            }
            .keyboardShortcut(.return, modifiers: .command)
            .disabled(state.isRunning || state.editorText.isEmpty)

            Button(action: { state.cancelRunningQuery() }) {
                Label("Cancel", systemImage: "stop.fill")
            }
            .keyboardShortcut(".", modifiers: .command)
            .disabled(!state.isRunning)

            if state.isRunning {
                ProgressView().scaleEffect(0.6).padding(.leading, 4)
            }

            Spacer()

            Button(action: { Task { await state.disconnect() } }) {
                Label("Disconnect", systemImage: "bolt.slash")
            }
        }
        .padding(.horizontal, 8)
        .padding(.vertical, 6)
    }

    /// Returns a reconnect closure only when the error looks like a connection-level problem.
    /// v0.2a keeps this deliberately simple: any error containing "connection closed" or
    /// "connection refused" gets a Reconnect button. More precise classification is v0.2b.
    private func reconnectIfConnectionLost(error: String) -> (() -> Void)? {
        let lower = error.lowercased()
        let looksConnectionLevel = lower.contains("connection closed")
            || lower.contains("connection refused")
            || lower.contains("connection reset")
        guard looksConnectionLevel else { return nil }
        return {
            // Reconnect = disconnect + user returns to ConnectSheet via RootView (Task 10).
            Task { await state.disconnect() }
        }
    }
}
```

```markdown
### Vendored C dependencies (conscious trade-offs)

LithePG authors zero C shims. Two upstream Swift packages we depend on ship vendored C that we accept deliberately:

1. **BoringSSL** via `swift-nio-ssl` — the TLS implementation behind `PostgresConnection`'s TLS path. Accepted in v0.1 for pinned-root trust behavior the Darwin SecTrust path wouldn't honor for internal CAs.
2. **tree-sitter** (core + `tree-sitter-sql` grammar) via `Runestone` — the incremental parser that drives the query editor's syntax highlighting from v0.2a onward. Accepted because rolling our own SQL tokenizer loses correctness on PG-specific syntax (dollar-quoted strings, `plpgsql`, array literals) that tree-sitter handles natively.

Both land at <5MB combined contribution to the release binary. The v0.4 <15MB budget still holds. Neither package accepts network input directly; both are used purely for local parsing/cryptography on data we already own.
```
