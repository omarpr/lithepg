# Historical plans and specifications

The files in this directory are archived implementation records. Commands and
artifact paths describe the repository as it existed during each milestone and
are not the current build, test or release instructions.

Use the root `README.md`, `CONTRIBUTING.md` and `docs/RELEASING.md` for current
commands. The milestone evidence log, evidence images and obsolete milestone
scripts referenced by some archived records were removed after the v1.0 source
release.
