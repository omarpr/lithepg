LithePG v1.0.0 — Public Launch

## GitHub release body draft

LithePG v1.0.0 is the public-launch release for a lean, Mac-native PostgreSQL client with local-first SQL drafting.

### Highlights

- Native macOS PostgreSQL client built in Swift with `postgres-nio` and no `libpq` dependency.
- Saved connection metadata with passwords stored separately in the macOS Keychain.
- Schema sidebar, renameable query tabs, query history, readable typed values, and a clickable results grid with copy and local cell inspection/editing.
- Result copy and export as CSV, TSV, JSON, GitHub-flavored Markdown or SQL INSERT statements.
- One-click table previews plus EXPLAIN and EXPLAIN ANALYZE plan trees with cost and timing summaries.
- A read-only schema graph with foreign-key edges, pan/zoom, deterministic large-schema layout and a column inspector.
- Connection-string paste cleanup and first-class Neon endpoint hints for direct and pooled URLs.
- A left-side connection navigator, an optional user-installed Neon CLI scanner for importing missing databases, and an in-form test-connection action.
- Ask-in-English SQL drafting that uses local schema context, inserts SQL for human review, and never auto-runs generated SQL.
- Light, dark, and system appearance settings, with dark as the default.
- Public project docs: changelog, security policy, governance, contribution guide, code of conduct, and issue/PR templates.
- A tested macOS packaging, signing, notarization and public-ZIP path with Homebrew cask metadata prepared.

### Local-first AI and privacy posture

LithePG does not send prompts, schema metadata, credentials, query history, or query results to cloud AI services. On supported macOS 26 Macs, Ask uses Apple's on-device system foundation model with bounded schema context, guided output and a read-only SQL safety gate. A deterministic local engine remains the fallback. LithePG bundles and downloads no model artifact.

Generated SQL is a draft: it is inserted into the editor for review and is not executed automatically.

### Known launch notes

- The app is macOS-only.
- AI model artifacts are separate user-provided downloads/experiments, not bundled assets.
- Index suggestions and table/column statistics, advanced AI beyond the current deterministic drafting scope, Parquet/Excel export, deeper appearance customization and auto-updates remain deferred.
- Public distribution is blocked until a Developer ID Application identity and notarytool profile are available. Source builds and the local ad-hoc candidate are development artifacts, not substitutes for the signed/notarized release.
